# Requirements

- pybind11
- Python 3
- OpenGL 4.5
- GLFW 3
- Ubuntu
- C++14 (gcc-6)
- Chainer 4

# Installation

**GLFW**

```
sudo apt install libglfw3-dev
```

**pybind11**

```
pip3 install pybind11 --user
```

**Renderer**

```
cd three
make
```

**imgplot**

```
cd imgplot
make
```

**Chainer**

```
pip3 install chainer cupy h5py
```
